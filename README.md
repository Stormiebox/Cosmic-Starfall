# Cosmic Starfall

[![License: CC BY-NC 4.0](https://img.shields.io/badge/License-CC%20BY--NC%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc/4.0/)

**Cosmic Starfall** is a modernization and continuation of the original Starfall concept for Avorion.

This project focuses on:
- anti-OP rebalance of high-impact systems,
- script QA hardening for reliability,
- and compatibility-minded evolution for broader Cosmic ecosystem usage.

## Quick Highlights

- Major rebalance pass on high-impact systems to reduce runaway dominance loops.
- Reliability fixes in high-risk scripts for safer long-session behavior.
- Added compatibility helper layer for optional ecosystem bridge usage.

---

## Installation

1. Place folder in:
   - **Windows:** `%AppData%\Avorion\mods\`
   - **Linux:** `~/.avorion/mods/`
2. Enable **Cosmic Starfall** in **Settings -> Mods**.
3. Restart Avorion when prompted.
